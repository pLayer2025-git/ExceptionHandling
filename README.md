# Exception Handling & Custom Exceptions in Java

This project demonstrates **robust exception handling** in Java, including the use of **try-catch-finally blocks**, **checked vs unchecked exceptions**, and **custom exceptions**.  

It showcases best practices for writing **exception-safe programs** with meaningful error messages and proper logging.

---

## **Features**

- Identify and handle **runtime exceptions** (e.g., `ArithmeticException`, `NullPointerException`).  
- Handle **checked exceptions** via **custom exception classes**.  
- Create and throw **custom exceptions** (`InvalidGradeException`).  
- Use **try-catch-finally blocks** for safe execution and cleanup.  
- Demonstrate **logging** of exceptions instead of printing stack traces.  
- Provide **meaningful error messages** to assist debugging.
